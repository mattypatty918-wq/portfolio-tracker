## 🔧 Technical Stack

### Frontend
- **Framework:** Preact (4KB - ultra lightweight)
- **Language:** TypeScript (full type safety)
- **Styling:** CSS 3 with CSS Variables
- **State:** Preact Hooks (useState, useEffect)
- **Build:** Vite (fast development, optimized production)

### Backend
- **Runtime:** Node.js
- **Pattern:** Serverless route handlers
- **Storage:** JSON files (SQLite/PostgreSQL ready)
- **API Style:** RESTful with query parameters

### DevOps
- **Version Control:** Git/GitHub
- **CI/CD:** Ready for GitHub Actions
- **Deployment:** Vercel, Docker, AWS Lambda, Traditional VPS
- **Monitoring:** Structured for Sentry, New Relic, DataDog

## 🚀 Quick Start

### Local Development
```bash
git clone https://github.com/mattypatty918-wq/portfolio-tracker.git
cd portfolio-tracker
npm install
npm run dev
# Open http://localhost:5173
```

### Deploy to Vercel (30 seconds)
```bash
npm i -g vercel
vercel
# Follow prompts - app is live!
```

### Deploy with Docker
```bash
docker build -t portfolio-tracker .
docker run -p 3000:3000 portfolio-tracker
```
