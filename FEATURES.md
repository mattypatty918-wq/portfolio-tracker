## Technical Implementation

### Frontend Stack
- **Framework**: Preact + TypeScript
- **Styling**: CSS Grid, Flexbox, Dark Mode
- **State Management**: Preact hooks (useState, useEffect)
- **Build**: Vite/esbuild compatible

### Component Architecture
```
App (Main router)
├── Dashboard (Overview)
├── Holdings (Stock positions)
├── Transactions (Buy/Sell/Dividend)
├── Analytics (Basic metrics)
├── AdvancedAnalytics (Phase 2)
├── PortfolioManager (Phase 3)
├── AdvancedPositions (Phase 5)
├── TaxReporting (Phase 6)
├── Reports (Multi-format export)
├── SecuritySettings (Phase 9)
└── Watchlist (Price tracking)
```

### Services
- **DataIntegration.ts** - API connectors, price fetching, data import/validation
- **Security.ts** - Authentication, audit logging, encryption, backup

### Data Structures
- Holdings: Symbol, Quantity, CurrentPrice, AvgCost, AssetType
- Transactions: Symbol, Type (buy/sell/dividend), Quantity, Price, Date
- Analytics: Returns, Volatility, Sharpe Ratio, Greeks, VaR

## How to Use

### Record a Transaction
1. Go to "Transactions" tab
2. Click "+ Record Transaction"
3. Fill in Symbol, Type (Buy/Sell/Dividend), Quantity, Price, Date
4. Click "Save Transaction"

### View Analytics
1. Go to "Analytics" for basic metrics
2. Go to "Advanced" for Sharpe Ratio, Sortino, VaR, etc.

### Manage Portfolio
1. Go to "Manager" to set target allocations
2. See rebalancing suggestions
3. View concentration risk and Kelly criterion position sizing

### Track Taxes
1. Go to "Tax" reporting
2. Download Form 8949 or Schedule D
3. Filter by year
4. Export capital gains report

### Advanced Positions
1. Go to "Advanced" to track Options, Crypto, Bonds, Margin
2. View options Greeks
3. Calculate staking rewards
4. Monitor leverage ratios

### Security
1. Go to "Security" settings
2. Enable 2FA
3. Create/restore backups
4. Export data (GDPR)
