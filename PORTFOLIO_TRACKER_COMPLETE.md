## 🎯 Architecture

### Frontend Stack
- **Framework:** Preact (lightweight React alternative)
- **Language:** TypeScript
- **Styling:** CSS 3 with CSS Variables (dark mode support)
- **State Management:** Preact Hooks (useState, useEffect)
- **Storage:** localStorage (UI state) + Backend API

### Backend
- **Node.js** route handlers in `/routes/portfolio.ts`
- **Persistence:** JSON file storage in workspace `/data/portfolio/`
- **API Pattern:** RESTful via `/v1/x/portfolio` endpoints
- **Resources:** holdings, transactions, watchlist, snapshots, reports

### Data Models
```typescript
// Holdings
{
  id, symbol, assetType, quantity, avgCost, currentPrice, 
  notes, tags, createdAt, updatedAt
}

// Transactions
{
  id, symbol, type (buy|sell|dividend), quantity, price, 
  date, notes, createdAt
}

// Watchlist
{
  id, symbol, assetType, addedAt
}

// Snapshots
{
  id, totalValue, totalCost, gainLoss, positions, recordedAt
}
```
